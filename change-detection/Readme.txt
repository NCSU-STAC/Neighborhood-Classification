Files in this folder can be used to detect changes between 2016 (new) data and 2002 (old) data.
detect_change_locations_openland.py - detect locations that were slum in 2016, but open land in 2002
detect_change_locations_urban.py - detect locations that were slum in 2016, but built-up area in 2002.
Please note, these results are dependent on accuracy of slum detection
